
# Installation

curl -LsSf https://astral.sh/uv/install.sh | sh

# Commands

[https://docs.astral.sh/uv/getting-started/features/](https://docs.astral.sh/uv/getting-started/features/)

```bash

uv init # Initialize a new project

uv add <package> # Add a package to the project
uv add -r requirements.txt # Add packages from a requirements file

uv sync # Sync the pyproject.toml file with the virtual environment
```
